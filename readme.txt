== Description ==
A thought after theme to empower blistering fast WordPress Charity, Organisation, Business, Entertainment, Tech, Personal brand blogs and Websites. It's built on top of Bootstrap to be responsive ready, it's smart to allow you change colors are you may want. It has a contact/inquirly form inbuilt. One column, Two-Columns, Three columns, centered and full-width layouts.

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Post Formats
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Excel Pearl in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
